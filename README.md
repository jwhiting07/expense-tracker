# Expense Tracker (Skeleton)

A full-stack project with **Spring Boot (Java)** backend and **React (Vite)** frontend.

## How to run

### Backend
```bash
cd expense-backend
./mvnw spring-boot:run
```
_Runs on http://localhost:8080_

### Frontend
```bash
cd expense-frontend
npm install
npm run dev
```
_Runs on http://localhost:5173_
